# huile8-ecdict
huile8 的离线词库（ https://github.com/mqycn/huile8 ）

## 使用方法

本项目 作为 [huile8](https://github.com/mqycn/huile8) 的附属，无法单独使用

```
# 1、代码克隆
git clone https://github.com/mqycn/huile8.git
cd huile8

# 2、更新词库
npm run dict.init
npm run dict.update

# 3、vscode 调试
使用 vscode debugger 进行测试 

# 4、打包，生成 vscode 插件
npm -g install vsce
npm run publish
```
